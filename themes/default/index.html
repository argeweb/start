Plugins

下列是預設已安裝於 argeweb 的套件, 你還可以透過 install.py 去安裝更多的組件

application_user

基礎的網站使用者, 具權限與身份機制.

backend_ui_material

Material 風格的網站管理後台, 使用 Daemonite Material UI 建立的, 子頁面是利用 iframe, 並加入了拖曳上傳, 排序等功能.

user_file

提供網站使用者進行檔案上傳, 上傳至 Google cloud storage, 並提供管理介面.

plugin_manager

可讓使用者自已決定是否啟用套件.

code

提供開發人員, 使其在線上編輯源始碼, 並為 css, js 提供壓縮版本
