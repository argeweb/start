Plugins

Below is the default plugin, installed in the initial package, you can use install.py to install more plug-ins

application_user

Basic web site users, with permissions and identity mechanisms.

backend_ui_material

Use Daemonite Material UI to build the backend management interface, Sub-page use iframe, and increase file drag and drop upload, sorting and more features.

user_file

Simplifies the process of uploading files to Google cloud storage, and provides a management interface on the backend.

plugin_manager

To manage all plugins In argeweb, you can enable or disable certain features at any time.

code

Provide developers with the ability to edit source code online and provide a compressed version of css, js
